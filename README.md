# 🦟 音波驅蚊 - Ultrasound Mosquito Repellent

> 使用超聲波干擾蚊子活動，一款瀏覽器-based 的驅蚊工具

## 📖 專案簡介

音波驅蚊利用 **Web Audio API** 產生 15-22 kHz 的高頻聲音，這些頻率人類大多數聽不見，但據說會干擾蚊子的活動行為。這是完全在瀏覽器中執行的輕量級工具，無需安裝任何 App。

## ⚙️ 功能特點

| 功能 | 說明 |
|------|------|
| **三種模式** | 單頻、雙頻交替、掃頻覆蓋 |
| **頻率調整** | 15-22 kHz 可調（0.5 kHz 步進） |
| **音量控制** | 0-100% 可調，中低音量即可 |
| **歷史記錄** | 最近 10 次使用紀錄（時間、模式、持續時間） |
| **Settings 持久** | 使用 localStorage 記住偏好設定 |
| **iPhone 風格 UI** | 大按鈕、圓角卡片、觸控友善 |

## 🔬 工作原理

### 超聲波與蚊子
- 蚊子翅膀振動頻率約在 150-500 Hz，可聽到較低的嗡嗡聲
- 某些超高頻聲音（>15 kHz）被研究顯示可能會干擾蚊子的導航系統或 mating behavior
- 本工具發射 **正弦波**（sine wave），因為弦波能量集中，對動物影響較單純

### 模式說明

1. **單頻模式**
   - 固定一個頻率（預設 17.5 kHz）
   - 適合多數蚊子
   - 可手動調整 15-22 kHz

2. **雙頻模式**
   - 在高、低兩個頻率之間交替切換
   - 切換速度可調（0.5-2 秒）
   - 增加不確定性，可能對不同品種更有效

3. **掃頻模式**
   - 頻率在選擇的範圍內上下掃描
   - 可設定最低與最高頻率
   - 掃描速度可調（0.5-5 秒/次）
   - 廣泛覆蓋，干擾效果最大化

## 📱 使用教學

### 第一步：開啟應用
在瀏覽器中訪問
### 第二步：選擇模式
在「頻率設定」區域選擇：
- **單頻**：通常就夠了
- **雙頻**：想提高覆蓋率
- **掃頻**：不確定哪種頻率有效時

### 第三步：調整參數
- 單頻：拖動滑桿選擇 kHz 數
- 雙頻：輸入兩個頻率，調整切換速度
- 掃頻：輸入最低/最高，調整掃描速度

### 第四步：設定音量
- 拖動音量滑桿，**建議 30-50%** 即可
- 音量過大可能造成耳鳴或頭暈
- 青少年與年轻人可能聽到的頻率較多，如果聽到尖銳聲請降低音量或停止

### 第五步：開始驅蚊
- 點擊 **▶ 開始驅蚊**
- 狀態指示器會變成綠色呼吸燈
- 將手機或喇叭放在蚊子活動區域
- **不要將耳朵靠近正在發音的裝置**

### 第六步：停止
- 點擊 **⏹ 停止** 關閉聲音
- 歷史記錄會自動存档本次使用

## ⚠️ 注意事項與警告

1. **效果不保證 100%**
   - 音波驅蚊的有效性仍有爭議，研究所得不一
   - 某些蚊子品種可能無反應
   - 效果可能因距離、障礙物而變化

2. **健康安全**
   - 避免長時間暴露（建議每 30 分鐘休息 10 分鐘）
   - 孕婦、耳疾患者應謹慎使用
   - 若出現頭暈、耳鳴、不適，請立即停止
   - **請勿在需要安靜的公共場所使用**（圖書館、會議室、醫院）

3. **設備注意**
   - 手機或筆電的喇叭可能無法有效播放 >20 kHz 的聲音
   - 建議使用外接喇叭或無線耳機（但耳機不要戴在頭上）
   - 部分低成本喇叭在超高頻時失真或功率不足

4. **法律與禮儀**
   - 在公共場所使用前請確認當地法規
   - 尊重他人，避免擾民
   - 夜間使用時注意音量

## 🐛 已知限制

1. **音響硬體限制**
   - 智慧型手機喇叭通常設計在 human hearing range (20 Hz - 20 kHz)
   - >18 kHz 的輸出功率可能很弱
   - 建議使用外接 USB 或藍牙喇叭

2. **時間精度**
   - `setInterval` 與 AudioContext 時間略有偏差，對驅蚊無明顯影響
   - 若切换标签页，浏览器可能 throttle 計時器

3. ** Battery 消耗**
   - 持續發聲會消耗電力，手機請接電源使用
   - 開啟计時器时建议连接充电器

4. **不適合的使用情境**
   - 嬰兒房、醫院、圖書館
   - 需極度安靜的場所
   - 自己需要專注思考時（高頻會干擾注意力）

## 🧪 測試方法

如果你想驗證聲音是否真的發出：

1. **使用頻譜分析 App**
   - 在手機上安裝 Spectrum 分析 App（如：Spectroid）
   - 將手機靠近播放設備，觀察頻譜是否在 15-22 kHz 有能量

2. **年輕人聽力測試**
   - 17 kHz 以上頻率，35 歲以下多數人還能聽到輕微高頻聲
   - 如果聽到「唧唧」聲，代表設備能輸出目標頻率

3. **蚊子行為觀察**（非嚴格）
   - 在封閉空間使用 30 分鐘
   - 觀察蚊子叮咬次數是否減少
   - 對照組（不開声音）比較

## 📚 參考資料

- 多篇實驗與系統性回顧（包含 Cochrane Review 類型研究）指出，市售超聲波驅蚊器在實際叮咬數下降方面 **整體效果有限或無顯著差異**，結果高度依賴環境與物種
- 部分實驗觀察到蚊子在特定高頻聲下會出現行為變化，但難以穩定轉化為「實用等級」的防蚊效果，因此建議僅作為 **輔助或實驗性工具**，勿取代蚊帳、防蚊液等傳統防蚊措施
- Web Audio API 文件：`https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API`
- 蚊子聽覺研究：蚊子可感知約 150–500 Hz 的翅膀振動聲，對更高頻的聲音也可能有生理反應，但目前對 >15 kHz 超聲波的行為影響仍在研究中
